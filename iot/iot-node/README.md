# IoT Node (ESP32) – Firmware

Tato složka obsahuje firmware pro IoT node (ESP32-S3) chytrého květináče.

## HW
- ESP32-S3-DevKitC-1
- SEN0308 (vlhkost půdy)

## Funkce
- čtení senzoru
- připojení k Wi-Fi
- pairing pomocí tokenu
- periodické odesílání dat na gateway / backend

## Jak spustit (placeholder)
- PlatformIO / Arduino framework
- konfigurace Wi-Fi a endpointů v `src/config.*`
